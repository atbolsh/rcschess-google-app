This page is one level of searching. Type in a set of parameters below, in
the first field, and a set of values in the second field, separated by a
semicolon and a space. The third field is all of the games included in the search
so far; feel free to remove any by hand.

The way that this works is that, if ANY of the parameters have ANY of the values,
the game will be returned in the search
results. For instance, if you put

White; Black

into the first field, and

Dennis Bolshakov; Luke Stephens

into the second, the search will return all games where EITHER Dennis OR Luke
play EITHER White OR Black; in other words, all games games played by Luke or Dennis.
If, instead, you want all games with Dennis as White AND Luke as Black, search first for
"White" and "Dennis Bolshakov", then use the "search through remaining" option on
the search page and search for "Black" and "Luke Stephens."

For your convenience, given is a list of common attributes in games.

White, Black, Event, Site, Date, Round, Result, PlyCount,
EventDate, TimeControl, ECO, Annotator, Opening.

This is not a comprehensive list.
